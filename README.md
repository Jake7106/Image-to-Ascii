# Image-to-Ascii
This Python script takes an image, and returns the image rendered in Ascii characters in .PNG and .TXT formats.
Currently, the application can only generate ASCII images for PNG, JPG, and JPEG files.

The script takes advantage of three Python libraries to function:
- os, which is included in Python's standard library
- PIL, which can be installed by running the command -> 'pip install Pillow'
- colorama, which can be installed by running the command -> 'pip install colorama'

How to run the script:
The script is simply run through the command line, below are the instructions to run the application.
- Navigate to project directory -> cd path/to/project
- Run the program -> python asciigen.py
- For macOS and Linux, you may need to run -> python3 asciigen.py
